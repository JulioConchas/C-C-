Lab - Titanium Basics

Mission: Your mission is to create a simple Titanium Mobile project and run it from Studio and the command-line interface (CLI).

Specification: You will create a default, "traditional" Titanium project and properly configure app properties. Then you'll run it from both Studio and the CLI.

To successfully complete this lab, you must develop an application that meets the following description:

  • App configuration: Name = HelloWorld, App ID = com.yourname.helloworld
  • The app runs without errors from Titanium Studio
  • You are able to run the app in the simulator/emulator by using the appropriate CLI command
1. In Studio, create a new Titanium Mobile project. In the project editor, enter the app name and app ID described above. Make the the appropriate target platforms are selected. Close the tiapp.xml.
2. Change the text of label1 to a string of your choice. Make sure that starting/ending quotes are present and match. Save your changes.
3. Build the project for the simulator/emulator available on your system. If you encounter any configuration problems, correct them by following the setup instructions on the documentation web site. Then, try running the project again.
4. Confirm that your NodeJS environment and the Titanium CLI are setup correctly:
  1. Open a terminal / Command Prompt window
  2. Enter titanium --version to view the CLI version.
  3. If you get an error, see the setup information at  Titanium Command Line Reference.
5. Using the reference information on the docs site, build and run the project using the appropriate CLI command. It will help to be in your project directory before running the command (that way, you won't have to supply the project_dir in the command). Observe the log information as the project builds.
6. Close the terminal window and the close the project in Studio.

Summary

In this lab, you created a default project and ran it using both Studio and the CLI. In this way, you confirmed that your system is configured correctly and you're ready to begin Titanium development.
