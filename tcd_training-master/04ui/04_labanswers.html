Lab - UI Deconstruction

Mission: This was a pen & paper exercise during which you determined the UI components needed to implement an app design.

Answers

Container components

  • One TabGroup to hold the two tabs
  • Two Tabs
  • Two Windows (one for each tab)
  • Two Tables (one for each window)

Visual (non-container) components

  • none at this stage of the app

Graphic assets

  • Window background graphic
  • Icon for each tab

Stylistic settings

  • Bar color (hex string)
  • Foreground (text) color for table, which also sets separator color
