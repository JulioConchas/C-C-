Lab - UI Deconstruction

Mission: This is a pen & paper exercise during which you will determine the UI components needed to implement an app design.

Specification: You will examine an app design and determine what UI components are going to be needed to create the user interface.

To successfully complete this lab, you must:

  • List all the container and visual UI objects that will be needed.
  • Determine what graphic assets will be needed to implement the app.
1. Examine the screenshot provided. Record a list of the container objects that are required for the app. List the visual (non-container) objects that will be needed. Determine the list of graphics needed (excluding splash screens, app icons, and app-store graphics).
2. Be prepared to share your lists with the instructor and other students in class. Update your lists if you missed any items.
3. Compare your lists with the answers provided at: S3: 04_labanswers.html

Summary

In this lab, you determined which visual and non-visual Titanium UI objects you will need to implement an app design. This exercise roughly mirrors what you will do as part of a real-world app development project (assuming that project follows the typical design/development workflow).
